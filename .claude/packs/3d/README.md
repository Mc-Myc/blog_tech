# Pack : 3d

Skills pour le temps réel 3D web (Three.js/WebGL). **Pas de rôle** : la 3D est
ici une compétence transverse (portée par le dev front ou le designer), pas un
métier à part dans le scaffold. Si un vrai rôle émerge à l'usage : `/create-role`.

| Élément | Type | Couvre |
|---|---|---|
| `skills/threejs-webgl/` | skill | fondamentaux et conventions Three.js |
| `skills/pipeline-assets/` | skill | de l'outil DCC au web : formats, compression, budgets |
| `skills/optimisation-scenes/` | skill | diagnostic et optimisation de performances |
